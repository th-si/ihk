Tools, die das Leben eines IHK Prüfers leichter machen sollen.
Ich Prüfe Fachinformatiker für Systemintegration an der IHK Dortmund und habe bei jedem Prüfungsdurchgang erneut Schwierigkeiten, mich in die Logik des Niederschrifsbogens einzudenken.
Daher habe ich einen Rechner geschrieben, der den Niederschriftsbogen im Layout nachahmt und so deutlich macht, welche Eingaben nötig sind und welche Werte an welchen Stellen der Niederschrift zu übertragen sind.

Die hier hinterlegten Tools setzen die Bewertungsbögen der [Handreichung der IHK zu Dortmund für IT-Berufe](https://www.ihk.de/blueprint/servlet/resource/blob/6164116/2d46bfb195c7ee193395213625ae6dc8/it-handreichungen-stand-mai-2024-data.pdf) sowie die Punktegewichtungen der [Ausbildungsordnung §§18-25](https://www.ihk.de/blueprint/servlet/resource/blob/4846794/908389a3b9f96bb396bef87363426618/verordnung-fachinformatiker-in-2020-data.pdf) um.

Alle hinterlegten Tools basieren auf HTML mit Inline Javascript, sodass alle Berechnungen lokal im Browser ausgeführt werden und die Tools portabel und unabhängig von einem Webserver sind.
Zum ausführen einfach das gesamte Projekt oder das jeweilige Tool herunterladen und mit einem beliebigen Browser ausführen.
Alternativ sind die Tools auch unter [nacke.it/ihk](https://nacke.it/ihk) aufrufbar.
